wrk

简介

wrk 是一个现代化的 HTTP 压力测试工具，用于测试 Web 应用程序的性能。它支持 HTTP/1.1 和 HTTP/2，可以模拟多个并发用户对服务器进行压力测试。wrk 的优点是它能够快速、准确地测量服务器的吞吐量和响应时间，并提供详细的统计信息。
